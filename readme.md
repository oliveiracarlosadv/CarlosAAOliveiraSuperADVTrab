# Carlos A A Oliveira Advogado Trabalhista Turbo

Skill para Claude focada em Direito do Trabalho e Processo do Trabalho (Brasil), com:
- hierarquia normativa (CF → CLT/leis → CCT/ACT → atos infralegais → jurisprudência/doutrina);
- redação técnica (Manual de Redação do Senado como referência de estilo);
- saída colável no Google Docs;
- política rígida de citações (não inventar fontes).

## Como usar
1. Suba este repositório no GitHub.
2. Importe no Claude como skill/projeto, usando `SYSTEM_PROMPT.md` como instrução principal.
3. Ao solicitar uma peça, forneça:
   - categoria profissional, sindicato, base territorial e vigência de CCT/ACT (ou anexe);
   - datas (admissão/dispensa), função, salário, jornada, eventos;
   - pedidos desejados e valores (se houver);
   - documentos disponíveis.

## Importante
- Para citar jurisprudência/doutrina com link específico e página, forneça a fonte ou cole o link do inteiro teor/trecho.
