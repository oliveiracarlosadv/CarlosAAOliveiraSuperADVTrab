# Política de citações e fontes (obrigatória)

## Proibição absoluta
- É proibido inventar: número de processo, link, data de consulta, páginas, relator, órgão julgador, ementa, trecho, DOI/ISBN.

## Jurisprudência — formato mínimo
**Tribunal:** [STF/STJ/TST/TRT-XX]  
**Processo:** [número completo]  
**Órgão julgador:** [Turma/Seção/Pleno]  
**Relator:** [nome] (se disponível)  
**Julgamento/Publicação:** [datas] (se disponíveis)  
**Link específico:** [URL do inteiro teor/acórdão específico]  
**Consulta em:** [data]  
**Trecho citado:** “...” (se disponível)

## Doutrina — formato mínimo
**Autor:** [nome]  
**Obra:** [título]  
**Edição:** [nº] — **Editora:** [nome] — **Ano:** [ano]  
**Página(s):** [p.]  
**Link específico:** [URL direta do trecho/visualização] ou **Identificador:** [ISBN/DOI]  
**Consulta em:** [data]

## Se faltar fonte
- Entregar apenas **fundamentação normativa** e **argumentação técnica**.
- Inserir campos “A PREENCHER” para o usuário colar as referências.
- Fazer no máximo **2 perguntas** para obter os dados faltantes.

## Preferência de fontes
- Priorize: sites oficiais de tribunais, diários oficiais, repositórios institucionais, editoras jurídicas reconhecidas.
- Evite blogs e agregadores não oficiais para citação formal.
